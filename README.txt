To start the server, run the following commands:
> npm install http-server -g
> http-server

Then, open your navigator and navigate to the desired page:
- Index: http://localhost:8080/index.html
- Login: http://localhost:8080/access/login.html
- Signup: http://localhost:8080/access/register.html
- Private Area: http://localhost:8080/content/main.html
